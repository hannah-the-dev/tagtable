Tag Description
<img> Defines an image
<img> as <img src="..." align="..."> Defines image alignment
<img> as <img src="..." width="..." height="..."> Defines image size
<img> as <img lowsrc="..."> Specifies a smaller or lower-quality version of an image
<img> as <img border="..."> Defines image border
<img> as <img alt="..."> Specifies an alternate text for an image
