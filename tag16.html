Tag Description
<form> Defines an HTML form for user input
<form> as <form method="..."> Specifies the HTTP method to use when sending getting form-data
<form> as <action="URL"> Specifies where to send the form-data when a form is submitted
<textarea> as <textarea cols="..."> Defines a multi-line text input control
<input> Defines an input control according to the type
<input> as <input type="radio"> Defines a radio button
<input> as <input type="checkbox"> Defines a checkbox
<input> as <input type="submit"> Defines a text field for entering a search string
<input> as <input type="reset"> Defines a reset button
<select> Defines a drop-down list
